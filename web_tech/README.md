Прогрмма лекций - стыд и позор(((  

Все примеры и упражнения пособия разделены на восемь разделов  
1.Язык гипертекстовой разметки HTML  
2.Каскадные таблицы стилей CSS  
3.Расширенный язык разметки XML  
4.Общий шлюзовый интерфейс CGI и язык Perl  
5.Объектные модели и языки скриптов VBScript и  JavaScript  
6.Технология активных серверных страниц ASP  
7.Технология PHP  
8.Включение и параметризация аплетов Java  


Для закрепления пройденного материала и его оценки в конце каждого раздела приводятся контрольные вопросы.  
Пособие предназначено для магистров по направлению 552800 “Информатика и ВТ” (специализация 552816 – “Информационное и программное обеспечения САПР”), изучающих дисциплины “Программирование для Интернет”, “Сетевые технологии”, “Объектно-ориентированное программирование”, а также для студентов специальности 2203 “Системы автоматизированного проектирования”.
